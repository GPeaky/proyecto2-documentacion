Instalacion de los Servidores y Cliente

Reto 3 - Fase 2

Servidor

Windows Server 2016

Para el proyecto utilizaremos el sistema operativo Windows Server 2016 debido al equilibrio entre la facilidad que ofrece la interfaz gráfica y la seguridad que nos proporciona. En la instalación.

Ubuntu Server 22.04

Utilizaremos Ubuntu Server como respaldo en caso de que el servidor principal, el Windows Server, falle. No realizaremos ninguna configuración específica.

Cliente

Windows 10

Utilizamos el cliente Windows 10 para realizar pruebas, ya que es muy fácil e intuitivo. De esta manera, podemos llevar a cabo pruebas de conexiones y otras acciones relacionadas con los servidores de manera rápida y eficaz.

Lubuntu

Utilizamos Lubuntu porque es la distribución que utiliza Labdoo. Además, es un sistema operativo de bajo perfil y muy accesible debido a que los requisitos mínimos son bastante bajos.
